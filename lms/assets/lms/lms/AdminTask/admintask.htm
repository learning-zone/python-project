CREATE USER GROUP
  1. Steps to create User Groups
    1. Enter User Group Name
    2. Select Modules & links for which rights has to be given to this particular group.
    3. Click on Add Rights to Create User Group with certain rights.
TOP
MODIFY USER GROUP
  1. Steps to modify User Group & reassign the rights for the selected group.
    1. Select User Group
    2. Uncheck the check box of links for which rights has to be removed.
    3. Click on Modify Rights to modify the rights for a particular group.
TOP
CREATE USERS
  1. Steps to create users.
    1. Enter User Name
    2. Enter Password
    3. Select User Group
    4. Select College
    5. Click on Add User to create the user.
TOP
MODIFY / DELETE USERS
  1. Steps to modify /delete users
    1. select user
    2. To modify user enter New Password & Confirm New Password and click update to change the user password.
    3. To delete user just select the user and click on delete button to delete the user.
TOP
ADD RIGHTS TO USERS
  1. Steps to add rights to users
    1. Select User
    2. Select Module as Main & submodule as Main and click on modules for which the rights will be assigned. This will be the main menu displayed when this particular user logs in to the Software. To remove the assigned right uncheck the check box and click update to remove the rights.
    3. To add individual links as rights to the user
    4. Select user, module, submodule. The list of links for this particular module & submodule will be displayed. check the checkbox to assign rights and click on update to add individual links rights to a particular user.
    5. To remove the assigned rights uncheck the check box and click update to remove the rights.
TOP
ADD RIGHTS - MARKS / ATTENDANCE
This is used to assign rights to a staff for a particular course & particular subject which he is handling. This is used restrict the staff by entering Marks & Attendance details of a student which he is not concerned with.
  1. Steps to add rights to a staff for entering marks / attendance details.
    1. Select Username / Staffname
    2. Select Course
    3. Click on subjects which the user is related to & click on Add Subjects to add rights.
    4. repeat this for adding subjects of different course.
    5. Click on Save Rights button to save the rights details of a staff.
TOP
DELETE RIGHTS OF STAFF
  1. Steps to delete rights of a staff.
    1. Select UserName / StaffName
    2. Check the subject links to delete the rights.
    3. click on Delete Rights to delete the rights for a staff.
TOP
VIEW LOG REPORT
This is used by administrator to check which user has logged in at what time & worked on which module.
  1. Steps to View Log Report.
    1. Select User
    2. Select Module
    3. Enter From Date & To Date
    4. click on View Detailed Log Report to view Log Report of the users.
TOP
VIEW LOG REPORT BY USERS
This report gives the details of user logins to the software.
  1. Steps to View Log Report By Users
    1. Enter From Date & To Date
    2. Enter Time
    3. select user
    4. click on View Log Report to view Log Report By Users of the users.
TOP
VIEW LOG REPORT BY MODULE

 

This is used to view the Log Report modulewise.
  1. Steps to View Log Report modulewise.
    1. Enter Log Date
    2. Enter Time
    3. Select User
    4. Select module
    5. click on Show Log Report to view Log Report modulewise.
