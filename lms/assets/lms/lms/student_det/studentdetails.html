USER RIGHTS
  1. Steps to add rights to users
    1. Select User
    2. Select Module as Main & submodule as Main and click on modules for which the rights will be assigned. This will be the main menu displayed when this particular user logs in to the Software. To remove the assigned right uncheck the check box and click update to remove the rights.
    3. To add individual links as rights to the user
    4. Select user, module, submodule. The list of links for this particular module & submodule will be displayed. check the checkbox to assign rights and click on update to add individual links rights to a particular user.
    5. To remove the assigned rights uncheck the check box and click update to remove the rights.

ADD SUBJECT RIGHTS TO STAFF
  1. Steps to add Subject rights of a staff.
    1. Select User / Staffname
    2. Select School Division
    3. Click on subjects which the user is related to & click on Add Subjects to add rights.
    4. repeat this for adding subjects of different School Division.
    5. Click on Save Rights button to save the rights details of a staff.

DELETE SUBJECT RIGHTS TO STAFF
